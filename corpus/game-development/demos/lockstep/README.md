# 락스텝 동기화 데모

스타크래프트 스타일의 결정론적 동기화를 직접 체험해보세요.

## 개요

이 데모는 락스텝(Lockstep) 네트워킹 방식을 시각화합니다. 두 플레이어가 동시에 게임을 진행하면서 네트워크 지연과 패킷 손실이 게임에 미치는 영향을 실시간으로 확인할 수 있습니다.

## 학습 목표

1. **락스텝의 동작 원리** - 모든 클라이언트가 같은 틱에 같은 입력을 처리
2. **입력 지연의 원인** - 네트워크 RTT가 입력 지연으로 변환되는 방식
3. **동기화 대기** - 패킷 손실 시 모든 플레이어가 기다려야 하는 이유

## 조작법

| 컨트롤 | 설명 |
|--------|------|
| 네트워크 지연 슬라이더 | 시뮬레이션할 RTT 설정 (0-500ms) |
| 패킷 손실 슬라이더 | 시뮬레이션할 패킷 손실률 (0-50%) |
| 플레이어 수 | 동시 플레이어 수 (2-4) |
