## 핵심 인사이트

선언형으로 더 높은 일을 하려면, 도구들이 계단식으로 쌓여야 한다.

## 원칙 구조

```
[메타 원칙] 계단식 진행
         │
    ┌────┴────┐
[구축]        [운영]
독자적 가치    위임 가능성
상호 발전      신뢰 기반 위임
```

### 메타 원칙: 계단식 진행
바로 안 되면 한 단계씩 쌓는다. 모든 원칙을 관통.

### 구축 원칙 (무엇을 만들 때)
- **독자적 가치**: 각 단계가 그 자체로 쓸모 있다
- **상호 발전**: 도구들이 서로를 고친다

### 운영 원칙 (어떻게 넘길 때)
- **위임 가능성**: 수동 → HITL → 기준 → Agent
- **신뢰 기반 위임**: 직접 해본 후에만 넘긴다

## 실제 사례: gemify ↔ forgeify 순환

```
gemify (지식 생산)          forgeify (코드 실행)
    │                            │
    ├─ improve-plugin 문서 ──────→ 코드 수정
    │                            │
    ←── 플러그인 검수/버그픽스 ────┘
                 현재 위치: 기준/규칙 축적 중
```

- 현재: Stage 1을 깊게 수행하며 Stage 2 진입 재료 수집
- capture-pair로 인사이트 포착 → 패턴 → 기준 추출 → 위임 가능
