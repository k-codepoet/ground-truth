# 플러그인 동기화 자동화 아이디어

## 핵심 인사이트

외부 표준(Claude Code Plugins 등)에 의존하는 플러그인은 **정기적 동기화**가 필수다. 수동으로 하면 누락이 생긴다.

## 자동화 기능 필요

- 공식 문서 변경 감지
- 현재 플러그인과 diff 비교
- 동기화 필요 항목 리포트

## 버전 호환성 정책 필요

**문제**: 무조건 최신화하면 하위 버전 사용자 호환 깨짐

**해결 방향**:
- 버전별 가이드 분기 (예: `skills/hook-guide/v1/`, `v2/`)
- 또는 가이드 내 버전 호환성 표기 (예: "v1.5+에서 지원")
- Claude Code 버전과 플러그인 가이드 버전 매핑

- 구체적인 자동화 방식 설계
- 버전 정책 결정 후 ced 플러그인에 적용
