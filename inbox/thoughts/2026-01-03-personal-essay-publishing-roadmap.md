# 개인 에세이 발행 로드맵

## 핵심 목표

나의 콘텐츠, 나의 이야기를 에세이/수필 형태로 작성하고 배포하는 시스템 구축

## 3단계 순차 진행

```
[1] 에세이 스타일 글쓰기 (콘텐츠)
         │
         ▼ 완료 후
[2] SSR 블로그 구축 (플랫폼)
         │
         ▼ 완료 후
[3] SNS 연결 (배포/홍보)
```

### Stage 1: 에세이 스타일 글쓰기 ← 현재 포커스

- 정제된 글쓰기 방법론
- 나의 관점, 나의 이야기
- library → views → export 파이프라인 활용

### Stage 2: SSR 블로그 구축 (Stage 1 완료 후)

- 메인 블로그 플랫폼
- SSR 기반 (기술 스택 미정)
- 개인 도메인

### Stage 3: SNS 연결 (Stage 2 완료 후)

- 블로그 콘텐츠 SNS 배포
- 홍보 채널 연결
