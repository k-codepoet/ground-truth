ced(claude extension dev) plugin에 기존 플러그인 기능을 개선하는 명령어/스킬 추가 아이디어.

**입력 유형**:
- 요구사항만
- 문제점만
- 요구사항 + 문제점

**프로세스**:
1. gemify처럼 되묻는 과정을 통해 목표 명확화
2. 개선 작업을 위한 문서 생성
3. 해당 문서 기반으로 실제 개선 작업 수행

**핵심**: 목표 불명확 → 대화로 구체화 → 문서화 → 실행
