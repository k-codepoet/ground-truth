# Ground Truth와 Plugin의 상호 시너지

> 날짜: 2025-12-31
> 출처: 대화

---

ground-truth에서 키우고 정리한 생각으로 knowledge-pipe-plugin을 만들 수 있다.

반대로, 플러그인들을 잘 만들면 ground-truth를 더 잘 만들고 관리할 수 있다.

지금도 claude skill, command로 이걸 관리하고 있음. 이게 잘 정리되면 plugin으로 만들면 됨.

**핵심 인사이트**: 양방향 시너지
- ground-truth → plugin: 정리된 지식이 플러그인 설계의 기반
- plugin → ground-truth: 플러그인이 지식 관리를 더 효율적으로
