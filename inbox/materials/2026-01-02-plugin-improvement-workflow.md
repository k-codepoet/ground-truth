# 플러그인 개선 워크플로우

## 실제 수행한 단계

### 1단계: 현재 가이드 로드
- `/ced:skill-guide`, `/ced:plugin-guide` 등 스킬 실행
- 플러그인 내 가이드 문서 내용 확인

### 2단계: 공식 문서와 비교
- WebSearch로 공식 문서 검색
- 각 가이드별 필드/이벤트 매칭 비교
- 누락/차이점 정리

### 3단계: 검증 결과 정리
- 일치하는 부분 ✅
- 누락된 부분 ⚠️
- 검증 필요한 부분 ❓

### 4단계: 수정 권장사항 도출
- 우선순위 결정
- 구체적인 수정 내용 제안

## 사용된 도구
- Skill: 가이드 로드
- WebSearch: 공식 문서 검색
- 비교 분석: 테이블 형태 정리

## 워크플로우 특징
- 자동화 가능한 반복 패턴
- 입력: 플러그인 경로
- 출력: 개선사항 리포트 + 수정 제안
