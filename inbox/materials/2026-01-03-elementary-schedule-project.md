# 초등학교 1학년 시간표 프로젝트 요약

## 프로젝트 배경
- 자녀가 2026년 초등학교 1학년 입학 예정
- 학교 제공 "학교 생활 안내" 문서 기반
- 맞벌이 가정에서 아이 오후 케어 시간 스케줄 작성 필요

## 핵심 데이터

### 기본 일과표 (1학년)
- 08:50 등교 완료
- 08:50-09:00 아침독서
- 09:00-12:10 1~4교시 (각 40분, 쉬는시간 10분)
- 12:10-13:00 점심시간
- 13:00-13:40 5교시 (화,수,목만)

### 요일별 하교 시간
- 월/금: 4교시 → 12:40 하교
- 화/수/목: 5교시 → 13:40 하교

### 방과후 프로그램 (총 40개, 1학년 신청 가능)

**카테고리별 분류:**
- 🎨 예술: 도자기, 미술(회화)
- 🧠 사고력: 바둑, 주산암산
- ⚽ 체육: 축구, 배드민턴, 방송댄스, 음악줄넘기, 퍼니스포츠
- 📚 언어: 독서논술, 한자, 영어그림책
- 🔬 과학: 로봇, 항공우주과학, 생명과학
- 🎵 음악: 바이올린
- 🍳 생활: 아동요리
- 💻 컴퓨터: 컴퓨터기초

**요일별 프로그램 수:**
- 월요일: 7개 (13:00~16:00)
- 화요일: 5개 (13:50~17:10)
- 수요일: 8개 (13:50~16:40)
- 목요일: 8개 (13:50~16:40)
- 금요일: 12개 (13:00~16:00)

## 기술 구현 현황

**완성된 기능:**
- React + TailwindCSS 기반 주간 시간표
- 방과후 프로그램 선택 (클릭 토글)
- 시간 충돌 감지
- 카테고리 필터
- 선택 프로그램 요약

**파일:**
- weekly-timetable-complete.jsx (인터랙티브 완성본)
